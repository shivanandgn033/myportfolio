Once the customer confirms the order and completes payment through PayPal, we generate a Ready To Print (RTP) file based on the job placed. Along with the RTP, our application generates a Job Description File (JDF) in XML format, which is used for interaction with the printer. The printer reads the JDF (XML) for job description, quantity, size, and orientation. After printing jobs are dispatched to the specified delivery address.
